11  安装 Python

11.1 介绍

在我们的学习序列中,您迄今大部分时间都在使用 Google Colab,这是一个便捷的基于云的环境,用于运行 Python 代码。现在是时候开始在您自己的机器上本地工作了。这允许您离线工作,并让您对开发环境具有更多控制权。

我们将引导您安装 Python 3.12.0,以确保与我们将要涵盖的示例和练习兼容。让我们开始吧!

在继续之前,我们将指示分为两个轨道:

  • Windows 用户
  • macOS 用户

请按照适用于您的操作系统的指示进行操作。

11.2 Windows 用户

11.2.1 步骤 1:检查您当前的 Python 版本

  • 在开始菜单中搜索“命令提示符”。
  • 在终端窗口中输入 python --version,然后按 回车。
  • 如果已安装 Python,它将显示版本号。
  • 如果您已经安装了 3.12.0 版本,您可以跳过其余步骤。如果您拥有其他版本(无论高低),请继续下一步安装 3.12.0。

11.2.2 步骤 2:下载 Python 3.12.0

  • 打开您的网络浏览器。
  • 前往 Google,并搜索 “Python 3.12.0”。
  • 在搜索结果中,找到提到 Python 3.12.0 的 python.org 链接。
  • 滚动到 Python 3.12.0 页面底部,找到 “Windows 安装程序(64 位)”。然后点击链接下载安装程序。

11.2.3 步骤 3:安装 Python

  • 找到下载的文件(通常在您的 下载 文件夹中)。
  • 双击安装程序以运行它。
  • 重要: 在安装程序窗口底部勾选 “Add Python 3.12 to PATH”。
  • 如果有选项,也请选择使用管理员权限。
  • 最后,您可能会被问及是否要禁用路径长度限制。也点击“是”。
  • 等待安装完成。

11.2.4 步骤 4:验证安装

  • 如果旧的命令提示符窗口仍然打开,请将其关闭。
  • 在开始菜单中再次搜索“命令提示符”并打开一个新的终端窗口。
  • 输入 python --version 并按 回车。
  • 您应该会看到 Python 3.12.0。 (如果您电脑上已经有 Python,可能会看到更新的版本。这没关系。从我们的 IDE 中,我们将能够选择刚刚安装的正确版本的 Python。)

11.2.5 步骤 5:在本地运行 Python

  • 在命令提示符中输入 python 并按 回车。
  • 在 >>> 提示符下,输入 2 + 2 并按 回车。
  • 您应该会看到 4。
  • 输入 exit() 并按 回车。

太好了! 您已成功安装 Python,并运行了您的第一个本地 Python 命令。

11.3 macOS 用户

11.3.1 步骤 1:检查您当前的 Python 版本

  • 前往 应用程序 > 实用工具 > 终端。
  • 输入 python3 --version 并按 回车。
  • 如果已安装 Python,它将显示版本号。
  • 如果您没有安装命令行开发工具,可能会弹出一个提示让您安装。请接受并安装。
  • 如果您的电脑尚未安装 Python,您会收到错误提示。
  • 否则,您应该会看到一个版本号。
  • 如果您已经安装了 3.12.0 版本,您可以跳过其余步骤。如果您拥有其他版本(无论高低),请继续下一步安装 3.12.0。

11.3.2 步骤 2:下载 Python 3.12.0

  • 打开您的网络浏览器。
  • 前往 Google,并搜索 “Python 3.12.0”。
  • 在搜索结果中,找到提到 Python 3.12.0 的 python.org 链接。
  • 确保 URL 来自 www.python.org,以避免非官方来源。
  • 滚动到 Python 3.12.0 页面底部。
  • 在 “文件” 部分下,找到 “macOS 64 位 universal2 安装程序”。
  • 点击链接下载安装程序。

11.3.3 步骤 3:安装 Python 3.12.0

  • 找到下载的文件(通常在您的 下载 文件夹中)。
  • 双击安装程序以运行它。
  • 等待安装完成。
  • 安装完成后,点击 “关闭”。

11.3.4 步骤 4:验证安装

  • 如果旧的终端窗口仍然打开,请将其关闭。
  • 再次前往 应用程序 > 实用工具 > 终端。
  • 输入 python3 --version 并按 回车。
  • 您应该会看到 Python 3.12.0。

11.3.5 步骤 5:在本地运行 Python

  • 在终端中输入 python3 并按 回车。
  • 在 >>> 提示符下,输入 2 + 2 并按 回车。
  • 您应该会看到 4。
  • 输入 exit() 并按 回车。

太好了! 您已成功在 Mac 上安装 Python 3.12.0,并运行了您的第一个本地 Python 命令。
