12  安装和使用 VS Code

12.1 介绍

在本课中,我们将探索 Visual Studio Code(VS Code),一款功能强大且多用途的编辑器,用于编写、运行和调试 Python 代码。由于其丰富的功能集和广泛的扩展库,VS Code 被广泛使用,使编码更加高效和愉快。

请注意,本课最好以视频形式学习,因为有许多图形用户界面步骤难以用文字描述。

12.2 学习目标

本课程结束时,您应该能够:

  • 打开 VS Code 并导航到编辑器、终端和资源管理器标签。
  • 创建一个新的 Python 文件,并使用 Python 扩展运行它。
  • 使用命令面板搜索并选择一个配色主题。

12.3 安装 VS Code

VS Code 可用于 Windows、macOS 和 Linux。在您喜欢的搜索引擎中搜索“VS Code”,然后访问官方网站 code.visualstudio.com。下载适用于您计算机操作系统的版本。

对于 macOS 用户,安装 VS Code 后,您可能需要将图标拖到“应用程序”文件夹中。

从“应用程序”文件夹中,打开 VS Code。

12.4 导航资源管理器标签

资源管理器 标签显示您当前工作空间中的文件和文件夹。当您第一次打开 VS Code 时,可能会提示您尚未打开任何文件夹。我们在 Python 中的大部分工作将组织在包含多个文件的文件夹(也称为工作空间)中。

让我们创建第一个工作空间:

  1. 在您的计算机上,导航到桌面或其他易于记忆的位置。
  2. 创建一个名为 first_python_workspace 的新文件夹。
  3. 在 VS Code 中,点击 “Open Folder” 按钮并找到您新建的文件夹。或者,您也可以将文件夹拖入 VS Code 窗口。

现在您的工作空间已在 VS Code 中打开,您可以创建一个新文件:

  1. 在 资源管理器 标签中,在工作空间文件夹内右击。
  2. 选择 “New File”。
  3. 将文件命名为 first_script.py。文件将自动在编辑器中打开。

12.5 编写和保存 Python 代码

让我们在新文件中编写一些 Python 代码:

print(2 + 2)

要保存文件,请按 Ctrl + S(或在 macOS 上按 Cmd + S)。

12.6 安装 Python 扩展

要在 VS Code 中运行 Python 代码,我们需要安装 Python 扩展:

  1. 点击左侧侧边栏中的 扩展 标签(看起来像四个方块)。
  2. 在顶部的搜索栏中,输入 “Python”。
  3. 找到由 Microsoft 发布的名为 “Python” 的扩展,然后点击 安装。

12.7 运行您的 Python 脚本

安装 Python 扩展后,您现在可以运行您的脚本:

  1. 如果 first_script.py 尚未打开,请打开它。

  2. 点击编辑器右上角的 运行 图标(播放按钮)。

    • 或者,您可以在编辑器中右击并选择运行 Python,然后选择 “Run Python File in Terminal”。

  3. 终端窗口将打开在屏幕底部,您的代码将执行。您应该会看到输出:

    4

12.8 使用命令面板

命令面板 是 VS Code 中一个强大的功能,允许您访问各种命令和设置:

要访问它,请点击 VS Code 窗口顶部的搜索栏,然后按 > 或选择“显示并运行命令”选项。

让我们练习更改配色主题:

  1. 输入 ‘theme’ 并选择 ‘Preferences: Color Theme’。
  2. 使用上下箭头键循环浏览可用的配色主题。
  3. 按 Enter 选择您喜欢的主题。

12.9 总结

在本课中,我们:

  • 安装了 VS Code 并设置了一个工作空间。
  • 创建并保存了一个 Python 脚本。
  • 安装了 Python 扩展 以启用运行和调试代码。
  • 使用了命令面板 来自定义编辑器的外观。

祝编码愉快!
